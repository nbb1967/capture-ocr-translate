# Capture-OCR-Translate

A simple utility for optical recognition and translation of text captured from the screen.

Automatic language detection for OCR and translation. Text only, no formatting.  
Launch the utility by clicking the tray icon or using hotkeys. 
Powered by [AutoIt](https://www.autoitscript.com/site/), [curl](https://curl.se/), [OCR.SPACE](https://ocr.space/) and [Google Translate](https://translate.google.com/).  
No installation required. Free. Provided "as is," without any warranties.
